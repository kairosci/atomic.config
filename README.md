# Kionite configuration

This repo include a simple config for optimize and manage **Fedora Kionite**

---

### manage_system.sh

This file clean pointless branch, delete un-necessary folder and upgrade packages with `rpm-ostree` and `flatpak` commands 

---

### set_rpm.sh

This file delete un-necessary packages and install several packages (with `rpm-ostree` command)

---

### set_flatpak.sh

This file delete un-necessary packages and install several packages (with `flatpak` command)
